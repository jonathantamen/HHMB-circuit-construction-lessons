# Interactive Circuit Construction Lessons

Author: Jonathan Tamen, founder @ [Helping Hands MB](https://helpinghandsmb.org/)

> [Helping Hands MB](https://helpinghandsmb.org/) is a 501(c)(3) non-profit organization dedicated to providing STEAM (Science, Technology, Engineering, Arts, and Math) education to younger students in order to improve their academic confidence.

This repository is to create an interactive lesson website on top of the [PhET Interactive Simulations, by the University of Colorado Boulder](https://phet.colorado.edu/sims/html/circuit-construction-kit-dc/latest/circuit-construction-kit-dc_en.html).

The lessons will be useful for students in the 4th and 5th grades, targeted for use in a summer camp or science lesson. The lessons will be created in a way that is easy to follow and understand, with clear instructions and examples.
